

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup



```bash
$ git clone https://github.com/Grupo-de-Estudos-AGES-Creatus/sprint5-backend.git

$ npm install
```

## Compile and run the project

```bash

# docker
$ docker-compose up -d

# db
$ npx prisma generate

# db
$ npx prisma db push 

# db
$ npx prisma db seed

# development
$ npm run start
```
