## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ git clone https://github.com/Grupo-de-Estudos-AGES-Creatus/projeto-final-backend.git

$ npm install
```

## Compile and run the project

```bash

# docker
$ docker compose up -d

# db
$ npx prisma generate

# db
$ npx prisma db push 

# db
$ npm run seed

# development
$ npm run start

# watch mode

```
